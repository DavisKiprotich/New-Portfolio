# New Portfolio

This repository contains the source code for my personal portfolio website.

## Overview

The portfolio presents my work as an Electrical and Telecommunications Engineer who also builds software. It brings together:

- Web development projects
- Mobile applications
- Power & IT (IoT) case studies
- Certifications and engineering background
- Contact and portfolio presentation content

## Stack

- React
- Sass
- React Icons
- React Hot Toast
- EmailJS

## Main Sections

- Hero
- About
- Core Focus
- Projects
- Tech Stack
- Certifications
- Experience
- Contact

## Run Locally

```bash
npm install
npm start
```

## Production Build

```bash
npm run build
```

## Notes

- Mobile apps include Play Store links and public source links where available.
- Power & IT (IoT) work is presented as case-study style project overviews.
- Certifications support both local certificate files and public verification links.
